on
Link copied to clipboard

Convenience method that will invoke the consumer on every event T created by EventSource.events.

The events are buffered in an unlimitedbuffer and launched in the supplied scope, which is me.schlaubi.lavakord.LavaKord by default. Each event will be launched inside the scope separately and any thrown Throwable will be caught and logged.

The returned Job is a reference to the created coroutine, call Job.cancel to cancel the processing of any further events.
